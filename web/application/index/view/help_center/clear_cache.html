如何清理缓存

除了用Ctrl+F5重新整理,还有手动清除浏览器缓存,以下版本不同针对自己浏览器版本操作

  • I8,IE9浏览器
  • 火狐浏览器
  • 谷哥浏览器

步骤一：在IE菜单点击 “internet属性”

步骤二：

  1. 点击“删除”按键
  2. 取消保留
  3. 选择删除的项目
  4. 点击“删除“

步骤一：菜单栏 工具 选择 清空最近历史记录

步骤二：单击“详细信息”

步骤三：选择要删除的内容，选择历史记录的时间，单击“立即清除”即可。”

步骤一：

  • 右侧板手图标
  • 选择 工具（tool）
  • 清空最近历史记录

步骤二：选择清除内容和时间，单击“清除浏览数据”即可。
